# Code of Conduct

## Standard

Be direct, respectful, and focused on improving SAGE for developers.

Do not harass, threaten, insult, dox, spam, or pressure maintainers or contributors. Do not post secrets, private logs, API keys, or other sensitive data in public issues or pull requests.

## Reporting

Report unacceptable behavior privately through the repository owner profile or GitHub moderation tools.

## Enforcement

Maintainers may edit, hide, delete, or lock comments and issues that violate this code. Repeated or severe violations may lead to a ban from the project.
