# SAGE Public Roadmap

## Current Focus

- Keep the public package CLI-first.
- Improve install trust and documentation.
- Expand token savings proof by AI agent/provider.
- Strengthen local firewall and policy controls.
- Keep dashboard proof aggregate-only.

## Near Term

- `sage savings` estimates by AI agent/provider.
- Team View preview in README; invite rollout after 100 users.
- Firewall status, rule, and audit commands.
- Better project-level reports.
- Exportable Markdown/CSV/JSON proof reports.

## Later

- Team Dashboard invite rollout after 100 users.
- Shared policies.
- GitHub Actions integration.
- GitHub bot comment generation.
- Agency proof reports.
- Public GUI release when stable.

## Non-Goals For The Public CLI Package

- Uploading raw command logs by default.
- Publishing source code or `.env` contents.
- Shipping the desktop GUI before it is stable.
