# v2.0.0-cli-public Release Notes

SAGE is now positioned as a public CLI-first package:

> SAGE is a local-first command wrapper for AI coding agents that compresses terminal output, preserves raw logs locally, and proves token savings with privacy-safe metrics.

## Install

```bash
pip install psycgod-sage
sage --version
sage connect
sage init
sage run -- python -m pytest
```

The PyPI distribution is `psycgod-sage`; the installed CLI command is `sage`.

## Included

- `sage run --` command wrapping
- Local raw log retention
- Compressed command summaries for AI context
- Local command history and proof metrics
- GitHub OAuth connection
- MCP config installation with `sage mcp install`
- Local dashboard server

## GUI Not Included Yet

The desktop GUI is not part of this public CLI package. `sage gui` prints public roadmap status instead of launching a desktop app.

## Known Limitations

- The GUI is not public yet.
- GitHub OAuth / SAGE API access is required for most API-backed commands.
- Telemetry level `0` is local-only; higher levels are opt-in and constrained by account/key policy.
- The public dashboard is aggregate-only and does not expose raw commands, raw outputs, file paths, or local logs.
- Local ML and agent behavior depends on local history volume and optional provider credentials.
