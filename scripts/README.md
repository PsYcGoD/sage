# SAGE Scripts

This folder contains local release/demo utilities only.

API connection is handled by SAGE machine authentication through:
SAGE machine authentication through:

```powershell
sage connect
```

Available helpers:

- `create_demo_gifs.py` - generate demo GIF assets.
- `create_demo_video_assets.py` - generate demo video assets.
- `create_team_preview.py` - generate team preview assets.
